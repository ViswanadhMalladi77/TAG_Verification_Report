# Kavach RFID — Tag Verification Dashboard

## Designed and Developed by Viswanadh Malladi

### HBL Engineering · Kavach Phase-I

A dark-themed industrial railway control room style dashboard for analysing Kavach RFID tag verification reports.

### Features
- Upload Excel report files and search sections
- View Tag Data Verification and Tag Placement Verification
- Multi-section comparison
- Live Progress Tracker connected to Google Sheets
- PDF export with division selection
- Fully responsive, works in Chrome and Edge

### How to Use
1. Open the dashboard: [https://viswanadhMalladi77.github.io/TAG_Verification_Report/](https://viswanadhMalladi77.github.io/TAG_Verification_Report/)
2. Upload your Kavach RFID verification Excel files
3. Search sections and view tag data
4. Click the Progress button for live section progress from Google Sheets
